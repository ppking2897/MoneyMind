## 產品概述

### 產品定位

AI 驅動的個人記帳應用，核心價值在於**降低輸入摩擦**。透過自然語言輸入和收據掃描，讓記帳變得像跟朋友說話一樣簡單。

### 目標用戶

- 想要追蹤支出但懶得填表單的人
- 重視隱私、不想連接銀行帳戶的用戶
- 想要簡單直覺的記帳體驗

### 差異化策略

- **本地優先 + 隱私導向**：資料存本地，不強制雲端同步
- **AI 驅動輸入**：自然語言 + OCR，非傳統表單
- **學習用戶習慣**：自動分類越用越準

### 核心驗證假設

> 用戶是否真的會用「說一句話」或「拍照」來記帳，而不是手動輸入？

## 快速開始

```bash

git clone <repo-url>

# 2. 設定 Gemini API Key
# 在 local.properties 加入：
GEMINI_API_KEY=your_api_key

# 3. 開啟 Android Studio 並執行
```
